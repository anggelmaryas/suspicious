Web-based Suspicious IP Logger

Aplikasi ini merupakan sistem pemantauan akses IP sederhana berbasis web yang dirancang untuk mendeteksi dan mencatat IP address yang mengakses sistem, lalu mengidentifikasi apakah IP tersebut mencurigakan atau tidak berdasarkan daftar blacklist lokal.
